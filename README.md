# Quick Notes

## Some dependencies

- flask
- Flask-SQLAlchemy
- flask-login
- Authlib
- requests
- Gunicorn

## Run Migrations

```sh
flask db init
flask db migrate
flask db upgrade
```

## Hosting

- [Render](https://render.com/)

## Database Provider

- Postgres through [neon.tech](https://neon.tech/)
